# Infrastructure

This repository contains the infrastructure as code (IaC) for Pukara. The code in this repository is used to automate the provisioning and management of our infrastructure resources, such as servers, networks, and storage, domains names, etc.

For the GitOps repository visit https://github.com/pukara-dev/gitops

NOTE: All documentation is located in our [internal developer portal](https://backstage.pukara.dev/docs/default/domain/platform)
